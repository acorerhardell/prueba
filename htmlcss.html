HTML/CSS

HTML es un lenguaje para el desarrollo de paginas de internet. Y CSS es un lenguaje para dar estilo a las paginas

En este proyecto se desarrollo una pagina web en donde estuviera todos los proyectos y unidades que se han desarrollado en la materia de Herramientas de Desarrollo de Software.

Las caracteristicas de este trabajo son:
  • Debe contener una pagina para cada proyecto y una pagina de inicio
  • Debe tener un encabezado,cuerpo y pie de pagina
  • cada pagina debe contener
    • un titular con el nombre del trabajo
    • un párrafo con la descripción del trabajo
    • una lista no ordenada con las características del trabajo
    • una o varias imágenes del trabajo realizado
    • un párrafo con la opinión personal acerca del trabajo

Opinion personal
